Repeatable Tasks Feature:
- Create tasks that repeat daily, weekly, or monthly
- Custom repeat patterns (e.g., every 2 days, every 3 weeks)
- Select specific days for weekly repeats (Mon, Wed, Fri, etc.)
- Optional end date for recurring tasks
- Automatic creation of next occurrence when completing recurring tasks
- Visual indicators showing which tasks are recurring
- Calendar view displays recurring tasks on all relevant dates

Calendar Enhancements:
- Long-press or double-tap any calendar day to create a task with that date
- Date is now automatically prefilled when creating tasks from calendar
- "Add task" button in empty calendar days
- Optimized priority indicators (2 bars max to prevent overflow)
- Individual task bars sorted by priority (high → medium → low → none)

User Experience:
- Improved calendar interaction with intuitive gestures
- Better visual hierarchy in calendar view
- Cleaner date selection workflow
- Enhanced task management with recurring patterns
